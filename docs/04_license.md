- `tcad/` Python scripts  
- `bsim/` analyzers & utilities

- Tutorials  
- Educational descriptions

# 4. CC BY-NC 4.0 (Figures & Diagrams)

- Educational diagrams  
- Block diagrams, architecture figures

- Modification

- Commercial use without permission
